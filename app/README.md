# Industrial AI Process Analyst

## Projektziel

Der Industrial AI Process Analyst ist eine Python-Anwendung zur Analyse industrieller Prozessdaten.

Langfristige Ziele:

- Datenimport (CSV, Excel, SQL, OPC UA, MQTT)
- Datenbereinigung
- Statistische Analyse
- Machine Learning
- Anomalieerkennung
- LLM-gestützte Analyse
- Automatische Berichterstellung
- Dashboard
